7.1. Basic Example¶

The following example shows basic usage examples

[1]:

import os

The example prints current directory (getchwd).

[2]:

def some_function():
    print(os.getcwd())

[3]:

if __name__ == "__main__":
    print("Current working directory:")
    some_function()
